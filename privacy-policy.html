Privacy Policy

Introduction

This privacy policy explains how the Sidebar with Gemini extension ("we", "our", or "the extension") handles your information. We are committed to protecting your privacy and being transparent about our data practices.

Data Collection and Use

We collect and use the following information:

  • Gemini API Key: Your provided API key is stored securely using Chrome's storage API and is used solely for authenticating requests to Google's Gemini service.
  • User Preferences: Settings such as sidebar visibility and theme preference are stored locally to maintain your chosen configuration.
  • Webpage Content: When you use the extension's features, the current webpage's content is temporarily processed to generate summaries and analysis. This content is never stored permanently and is discarded after processing.

Data Storage and Security

All data is stored locally on your device:

  • API keys are securely stored using Chrome's sync storage
  • User preferences are saved in local storage
  • No data is stored on external servers
  • All communication with the Gemini API uses secure HTTPS connections
  • We implement security best practices to protect your data

Third-Party Services

The only third-party service we interact with is Google's Gemini API:

  • Webpage content is sent to Gemini API for processing
  • Your API key is used for authentication
  • No personal information is shared with any other third parties
  • We do not use analytics or tracking services

Data Removal

You can remove your data at any time by:

  • Using the "Clear API Key" button in the sidebar
  • Clearing the extension's storage through Chrome's settings
  • Uninstalling the extension (removes all stored data)

Required Permissions

The extension requires the following permissions to function:

  • activeTab: To access the current webpage for analysis
  • scripting: To inject the sidebar interface
  • storage: To save your API key and preferences
  • tabs: To manage the sidebar across different tabs
  • Host Permission (generativelanguage.googleapis.com): To communicate with the Gemini API

Children's Privacy

The extension is not intended for use by children under 13 years of age. We do not knowingly collect personal information from children under 13.

Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new policy in this location.

Contact Us

If you have any questions about this privacy policy or our data practices, please contact us through our GitHub repository.
